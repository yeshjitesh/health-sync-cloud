# DVDS-Health

AI-powered health management platform for the Diu Vanja Darji Samaj community.

## Features

- **DVDS Bot** - AI health assistant with region-specific guidance (NHS, CDC, ICMR, WHO)
- **Disease Predictor** - Risk assessments for diabetes, heart disease, kidney, and liver conditions
- **Health Records** - Track vital signs, lab results, and health metrics
- **Medication Tracker** - Never miss a dose with smart reminders
- **Multi-Language Support** - Available in English, Gujarati, and Portuguese
- **PWA Support** - Install as an app on mobile devices

## Technology Stack

- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (Database, Auth, Edge Functions)
- Framer Motion for animations

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
